# 佘诗曼 · 决策与行动记录

## 研究重点
关键职业决策：角色选择、市场转型、合约决定、人生转折点的决策逻辑。

---

## 决策 1: 参选香港小姐（1997年）

- **时间**: 1997年
- **决策**: 报名参加香港小姐竞选，最终获季军
- **背景**:
  - 佘诗曼在瑞士读完酒店管理课程回港，四处找工作
  - 在瑞士读书时，她有家庭条件优渥的男友，对方可以照顾她，不用工作。当时她心想：**"我在干什么？我才22岁，不甘心这样一辈子。"** 于是分手、回港
  - 回到香港后，电视里传来港姐比赛的音乐，在母亲支持下报了名
- **公开阐述的理由**: 不甘心安逸但被动的人生，想要自己掌控方向
- **观察到的决策模式**: 这是她人生中第一次重大"不甘心"驱动的决策。她后来的所有重大决定都遵循同一模式 — **拒绝被动安排，主动选择不确定性**
- **事后意义**: 获季军后签约TVB，正式进入演艺圈。这是一个非传统的入行方式（她没有任何表演经验或兴趣），但成为她整个职业生涯的起点

## 决策 2: 在"鸡仔声"批评中选择坚持（1997-1999年）

- **时间**: 1997-1999年
- **决策**: 在全港嘲笑声中不退出，选择正面突破
- **背景**:
  - 首部电视剧《明天不一样》，她完全不会演戏，连哭戏都哭不出来
  - 拍《雪山飞狐》时被全港骂"鸡仔声"（声音小而尖细），被称为"最丑港姐"
  - 有工作人员当众嘲讽：**"街边卖鱼蛋的阿婆都比她自然。"**
  - 她曾想过：**"不如我退出娱乐圈"**
- **公开阐述的理由**:
  - 她用行动而非言语回应：把所有批评她的报纸剪下来，贴在镜子上每天激励自己：**"我以后不想再看到这些声音，佘诗曼，你给我做好！"**
  - 问前辈黄日华怎样才能演得更好，对方说"多演就好"
- **观察到的决策模式**: **最笨拙也最扎实的方式** — 不走捷径，用最笨的方法（每天练）来突破弱点。后来她离开TVB北上、疫情期间上网课，都遵循同样的"苦练突破"模式
- **事后意义**: 她的"不服输"内核被彻底点燃，成为她此后所有行为模式的原点

## 决策 3: 每天凌晨四点读报纸练声（约2000年）

- **时间**: 约1998-2000年间
- **决策**: 每天在家大声读报纸练声、练咬字、练气息
- **背景**:
  - 被批评"鸡仔声"后，她到处请教前辈台词技巧
  - 导演楚原先生给她出主意："不然你在家大声骂脏话吧？" 她担心被邻居投诉。楚原说："那你拿报纸读吧，大声读出来。"
- **公开阐述的理由**: 要把最被嘲笑的短板变成自己的强项
- **观察到的决策模式**: **针对弱点精准突破**。她后来为演内地剧请老师逐句教普通话，为演法官去法庭旁听，都遵循同一模式 — 找到最薄弱的环节，用最直接的方法攻克
- **具体行动**:
  - 每天凌晨四点起床读报练声
  - 洗澡时也在背台词
  - 一遍遍看经典影片，从模仿到自主表演
  - 对着镜子观察表情
  - 精神高度集中，没日没夜背台词
- **事后意义**: 2000年《十月初五的月光》中她出场即一大段个人台词，说得抑扬顿挫、字字到位。**"那是我第一部没有被骂的戏。"** 该剧成为TVB年度收视冠军，她第一次入围"视后"

## 决策 4: 争取《金枝欲孽》反派角色尔淳（2004年）

- **时间**: 2004年
- **决策**: 主动请监制瞒着公司老板，"骗"到一个反派角色
- **背景**:
  - 2000年后她逐渐成为TVB当家花旦，角色都是正派大女主
  - 她深知这样下去会越走越窄：**"角色都是正派大女主，她深知这样下去会越走越窄"**
  - 无奈电视台为了维护她的形象就是不给反派角色
  - 最后她请监制瞒着老板，才"骗"到这个反派
- **公开阐述的理由**: 不愿被固定在舒适区，主动打破自己的形象
- **观察到的决策模式**: **主动打破舒适区** — 她后来接《延禧攻略》女配角（被认为"自降咖位"）、接从未尝试过的律政题材，都遵循同一模式。她对标签的态度是：**"不要沉在这个标签里面，你要把它扔掉"**
- **事后意义**:
  - 当时付出代价：有厂商因她演反派而撤回嫦娥代言（"没有嫦娥这么凶、这么狠的"）
  - 但《金枝欲孽》在全亚洲掀起热潮，尔淳成为经典角色
  - 她第一次学到什么是"收着演"，演技进化的关键节点

## 决策 5: 凭《凤凰四重奏》首夺视后（2006年）

- **时间**: 2006年
- **决策**: 在《凤凰四重奏》中一人分饰四角（跨越四个时代的女性）
- **背景**: 经过多年磨练，她已经是TVB一线花旦，但尚未获得"视后"认可
- **公开阐述的理由**: （这是她努力的成果而非一个选择决策，但获奖感言反映了她的心态）
  - 获奖感言：**"由那些人话我鸡仔声，唔识做戏，直到我今日攞这个奖，好开心，我没令到支持我的人失望，没令我屋企人、没令我妈妈失望。"**
- **观察到的决策模式**: 她把这个奖看作"证明" — 不是证明自己比别人强，而是证明"没有辜负"支持她的人
- **事后意义**:
  - 成为TVB首位"双料视后"（最佳女主角 + 我最喜爱的电视女角色）
  - 站上第一个人生高峰，但随即陷入迷茫：**"你不会永远在高峰的。下来以后怎么办？"**

## 决策 6: 结束TVB管理合约，北上发展（2011年）

- **时间**: 2011年
- **决策**: 约满后不续约TVB，独立发展，主攻内地市场
- **背景**:
  - 2006年获视后后达到高峰，但感到迷茫
  - 她回忆当时的想法：**"不踏出那一步，我永远不会知道未来怎样，我觉得要跳出温室，尝试走一条自己的路。"**
  - TVB是她出道14年的"娘家"，是舒适区
  - 暂别时，全体剧务请她吃饭送行，她哽咽回忆：**"他们真的好像我爸爸一样。"**
- **公开阐述的理由**: 要跳出温室，走自己的路
- **观察到的决策模式**: **在高峰期主动离开舒适区**。她在"温室"里是女王，但她选择主动跳出去。后来她从TVB到内地，从正剧到反派，从女主到女配，每一次都是在安全的时候选择冒险
- **事后意义**:
  - 起初遭遇低谷：**"当我决定离开TVB时，我以为一样会有很多工作，不用担心将来，但实情是少了人会找我，因为他们不知道怎样去联络我。"**
  - **"我以后是不是完蛋了呢？"** — 她曾自我怀疑
  - 但她用这两年"扩容"：大量观摩学习，积累"资料库"
  - 这段低谷最终成为她后来爆发的跳板

## 决策 7: 人生哲学转变 — 母亲患病的冲击（约2024年）

- **时间**: 2024年前后（确切时间未公开）
- **决策**: 为照顾患病的母亲，调整工作重心回香港
- **背景**:
  - 母亲确诊患癌（佘诗曼未公开否认），接受高强度治疗
  - 佘妈妈曾剃光头，展现极强韧性
  - 佘诗曼说：**"妈妈之前是不舒服，她心态好好，好乐观正面，接受适当治疗。"**
  - 父亲在她5岁时因车祸离世，母亲独自抚养三个孩子长大
  - 近两年，与她情同父女的演艺前辈许绍雄离世
- **公开阐述的理由**:
  - 第四次获奖感言中感谢弟弟：**"好多谢我细佬，好悉心去照顾妈妈，令到我可以好任性这么去追寻我的梦想"**
  - 她将"追寻梦想"称为"任性"，说明她内心清楚家庭责任的分量
- **观察到的决策模式**: 务实地平衡责任与梦想，不是二选一，而是找到支持系统（弟弟照顾母亲）来让自己能继续工作
- **事后意义**:
  - 人生更加通透：**"生离死别是必修课，重要的是学会'修复自己，勇敢地面对'。"**
  - 公开已立遗嘱，以确保家人的生活
  - 这种"务实面对死亡"的态度，与她母亲的影响一致

## 决策 8: 凭《使徒行者》二夺视后（2014年）

- **时间**: 2014年
- **决策**: 接拍《使徒行者》，饰演"钉姐"（爱贪小便宜又正直坚强的卧底）
- **背景**:
  - 离开TVB后北上发展，已经适应了内地市场
  - 这次是回归TVB拍剧
  - "钉姐"与她之前的角色很不同 — 不是正派大女主，而是有小毛病的市井人物
- **公开阐述的理由**: （从她的整体角色选择逻辑推断）喜欢角色的复杂性
- **观察到的决策模式**: 选择有反差感的角色。从正派到反派（尔淳），从女主到女配（娴妃），从女王到市井（钉姐），她总是在挑战观众对她的固有印象
- **事后意义**:
  - 再次拿到TVB视后，成为TVB台庆历史上获奖最多的演员
  - 钉姐成为又一个经典角色
  - 获奖感言中的座右铭：**"没嘢系最好，只有更好"**

## 决策 9: 接演《延禧攻略》反派娴妃（2018年）

- **时间**: 2017年接拍，2018年播出
- **决策**: 接演女配角娴妃（后黑化的反派），进入内地市场
- **背景**:
  - 离开TVB后经过两年低谷期，大量学习积累
  - 《延禧攻略》一开始并不被看好
  - 她拿到的是女配角，被认为"自降咖位"
  - 但她看了剧本后迅速决定：**"我看剧本的时候就决定出演了，因为我只花了不超过3天看完了70多集剧本，停不了。"**
  - **"再经历一两年的彷徨，我有机会接到《延禧攻略》的演出，我就决定要好好珍惜这个机会努力做好它。"**
- **公开阐述的理由**:
  - 剧本质量是唯一判断标准，不看番位、不看市场预期
  - 这是她"信自己"判断力的典型体现
- **观察到的决策模式**: **用直觉+快速行动做出决定**。她不是反复犹豫型的人 — 好的剧本3天内决定，好的角色一看到就想接。但这个"直觉"背后是她几十年积累的专业判断力
- **事后意义**:
  - 她扮演的娴妃被观众称赞：**"不需要烟熏妆就能完成黑化"**
  - 《延禧攻略》成为现象级爆款，彻底打开内地市场
  - 证明她低谷期的"扩容"策略是正确的
  - 这是她"人生曲线"中最漂亮的一次低谷反弹

## 决策 10: 回归TVB拍《新闻女王》（2023年）

- **时间**: 2022-2023年拍摄，2023年底播出
- **决策**: 距离上次拍TVB剧6年后，回归"娘家"拍《新闻女王》
- **背景**:
  - 2017年后她一直在内地发展
  - 《新闻女王》由优酷与TVB联合出品，是合拍剧
  - 她说：**"我真的好感激，如果TVB没有给这么多机会栽培我，我不会今日。"**
  - 她将回归形容为"好像回家一样，所有工作人员我都认识的"
- **公开阐述的理由**:
  - 赶上了港剧最"丰盛"的时代
  - 剧本质量打动她
  - 她提前五个月潜入TVB新闻部实习，将历年事故数据背得滚瓜烂熟
- **观察到的决策模式**: **战略性回归** — 不是退回舒适区，而是带着新积累回到老战场，用更高的水平重新定义自己的位置
- **事后意义**:
  - 《新闻女王》同时在内地和香港爆红
  - 在网络播出平台优酷创造多个数据纪录，每天收割数个热搜
  - 在TVB收视率第二周便达27.9%
  - 她凭此第三次获TVB视后，成为TVB首位三封视后
  - 《新闻女王》成为现象级作品，"Man姐"成为文化符号
  - 接下来的《新闻女王2》让她四封视后，创造TVB历史

## 决策 11: 四封视后 — 历史性成就（2024/2026年）

- **时间**:
  - 2024年1月：凭《新闻女王》第三次获TVB最佳女主角（TVB首位三封视后）
  - 2026年1月：凭《新闻女王2》第四次获TVB最佳女主角（TVB开台58年首位四封视后）
- **背景**:
  - 她此前已于2006年、2014年两次获TVB视后
  - 2024年的三封已是历史纪录
  - 2026年的四封彻底打破自己的纪录
- **获奖感言中的核心理念**:
  - 2024年：**"我可以证实'一不离二、二不离三'都可以是好嘢！"**
  - 2026年：**"上一次拿奖，我觉得是佘诗曼拿的，但这次，我觉得是Man姐拿的。我不只是饰演这个角色，而是成为这个角色。"**
  - 2026年：**"历史是人类创造出来，只要没放弃，你们一样会成为历史。"**
- **观察到的模式**: 四次获奖感言构成了一个完整的成长弧线 —
  1. 2006年：**证明**（我没让大家失望）
  2. 2014年：**精进**（没有最好，只有更好）
  3. 2024年：**共荣**（这不是我一个人的努力）
  4. 2026年：**传承**（创造历史，激励后辈）
- **事后意义**:
  - 成为TVB开台58年来首位四封视后的演员
  - 她的角色（钉姐、娴妃、Man姐）跨越了港剧黄金时代到合拍剧新时代
  - 2026年3月，她携律政剧《正义女神》回归，继续拓展角色边界

## 决策模式总结

### 恒定模式（贯穿所有决策）

1. **"不甘心"驱动**: 每一次重大决定都源于对现状的不满足 — 不甘心做富太太、不甘心被叫"鸡仔声"、不甘心只演正派、不甘心留在温室
2. **在安全时选择冒险**: 她从不在危机中仓促决定，而是在稳定时主动跳出 — 高峰期离开TVB、女王期接女配角、安全时挑战新题材
3. **用最笨的方法突破**: 读报纸练声、请老师逐句教普通话、去法庭旁听、潜入新闻部实习 — 她不用聪明的方法走捷径
4. **直觉快速决策 + 苦练执行**: 决定往往在几天内做出（3天看完70集剧本决定接《延禧》），但准备过程极其漫长和扎实
5. **低谷转化为增值期**: 每一次低谷她都不沉溺，而是迅速转向"填满自己" — 观摩学习、上网课、请教前辈
6. **重情义不重利益**: 离开TVB时对剧务大哥的感恩、接角色时看剧本不看番位、颁奖时反复感谢团队而非炫耀自己

### 底层逻辑
**"信自己"不是一个口号，而是一个操作系统** — 她的每一步都是在不确定中选择相信自己能行，然后用"最笨的方法"把这个"信"变成现实。
